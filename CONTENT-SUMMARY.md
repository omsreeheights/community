## 🚀 How to Customize

### Update Community Information
Edit in `config.js`:
```javascript
const CONFIG = {
    siteName: 'Om Sree Heights Community',
    cooperativeName: '...',
    totalFlats: 600,
    location: '...',
    managementEmail: 'management@omsreeheights.com',
    // etc.
};
```

### Add New Announcement
Edit `config.js` announcements array:
```javascript
announcements: [
    {
        id: 9,
        date: 'Mar 25, 2026',
        title: 'Your Announcement',
        description: 'Details here...'
    },
    // ...
]
```

### Add New Event
Edit `config.js` events array:
```javascript
events: [
    {
        id: 11,
        month: 'Jun',
        day: '15',
        title: 'Event Name',
        description: '...',
        time: '...',
        location: '...'
    },
    // ...
]
```

### Update Board Members
Edit `config.js` board object:
```javascript
board: {
    chairman: {
        name: 'Your Name',
        position: 'Chairman',
        contact: '9876-XXXXXX'
    }
}
```
