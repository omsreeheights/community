## 📝 How to Update Content

All website content is managed through a single file: `config.js`

**To make changes:**
1. Open `config.js` in any text editor
2. Update announcements, events, contact info, etc.
3. Save the file
4. Commit and push to GitHub
5. Website updates automatically within 1-2 minutes

**No HTML or CSS knowledge required!**
