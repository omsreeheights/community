# Contributing to Om Sree Heights Community Portal

We welcome contributions from community members! Here's how you can help:

## How to Contribute

### 1. Reporting Issues
- Report bugs or feature requests via GitHub Issues
- Include clear descriptions and steps to reproduce bugs
- Provide screenshots for UI issues

### 2. Submitting Changes

**Fork and Clone:**
```bash
# Fork the repository on GitHub
# Clone your fork
git clone https://github.com/yourusername/community.git
cd community
```

**Create a Branch:**
```bash
git checkout -b feature/your-feature-name
# or
git checkout -b fix/bug-description
```

**Make Changes:**
- Edit files as needed
- Test locally before committing
- Follow code style conventions

**Commit and Push:**
```bash
git add .
git commit -m "Clear description of changes"
git push origin feature/your-feature-name
```

**Submit Pull Request:**
- Go to GitHub and create a pull request
- Provide detailed description of changes
- Reference related issues if applicable

## Code Style Guidelines

### HTML
- Use semantic HTML5 elements
- Indent with 4 spaces
- Keep lines under 120 characters
- Use lowercase for tags and attributes

### CSS
- Use meaningful class names
- Organize styles logically (layout, typography, components)
- Use CSS custom properties for colors
- Comment complex sections

### JavaScript
- Follow ES6+ standards where possible
- Use meaningful variable names
- Add comments for complex logic
- Use console.log sparingly (only for debugging)

## Testing Before Submission

1. **Local Testing:**
   - Test on desktop browsers (Chrome, Firefox, Safari)
   - Test on mobile devices or use DevTools
   - Test all forms and interactive elements
   - Test tab navigation

2. **Code Review Checklist:**
   - [ ] No console errors
   - [ ] Responsive on mobile/tablet/desktop
   - [ ] Accessibility maintained
   - [ ] Links and forms work correctly
   - [ ] Images load properly

## Contribution Areas

### We're Looking For Help With:
1. **Content:** Event descriptions, photos, newsletters
2. **Features:** New sections or improvements to existing ones
3. **Design:** UI/UX improvements
4. **Documentation:** Better guides and tutorials
5. **Bugs:** Bug fixes and optimization
6. **Translations:** Multilingual support

### Priority Contributions:
- Real community photos (with permission)
- Event management system enhancements
- Admin panel for notice board updates (Phase 2)
- Newsletter content automation
- Emergency notifications feature

## Guidelines

1. **Be Respectful:** Treat all contributors with respect
2. **Be Constructive:** Provide helpful feedback and suggestions
3. **Follow Best Practices:** Maintain code quality and standards
4. **Document Changes:** Update README if needed
5. **Test Thoroughly:** Ensure your changes don't break existing features

## Community Standards

- Be welcoming and inclusive
- Focus on what's good for the community
- Respect privacy and permissions (especially for photos)
- No harmful or inappropriate content

## Questions?

- Open an issue with your question
- Check existing documentation first
- Ask community managers directly

Thank you for helping improve Om Sree Heights Community Portal! 🙏
